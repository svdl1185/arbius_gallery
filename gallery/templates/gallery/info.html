What is This Gallery?

This gallery showcases images generated through the Arbius protocol - a decentralized AI network running on the Arbitrum blockchain. Each image represents a unique AI computation performed by distributed miners around the world.

How It Works

1

Request Submitted

A user submits an AI image generation request to the Arbius protocol, including their prompt and payment.

2

Miner Processing

Decentralized miners compete to fulfill the request using their AI models and computing power.

3

Result Submission

The winning miner submits the generated image and receives payment through the blockchain.

4

IPFS Storage

The image is stored on IPFS (InterPlanetary File System) for decentralized, permanent access.

5

Gallery Discovery

Our system automatically discovers and displays new images by monitoring blockchain events.
